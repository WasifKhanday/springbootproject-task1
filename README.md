# First Spring App

## Technologies
- Java 17
- Spring Boot 3.5.14
- Maven
- Thymeleaf

## Endpoints

### GET /
Returns plain text response.

### GET /greeting
Returns HTML page with dynamic greeting and image.

Example:
http://localhost:8080/greeting?name=Vistula

## Features
- Spring MVC
- Thymeleaf templates
- GET requests
- Query parameters
- Static image support
